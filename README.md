# fn60

fn60: use fn key to extend 60% keyboard easily

- `capslock` → `ctrl`
- key to the right of `space` → `fn`
    
    
    | fn +  | → |
    | --- | --- |
    | j l k h | ▾ ▸ ▴ ◂ |
    | , . | page up page down |
    | [ ] | home end |
    | 1 .. 9 0 - =  | f1 .. f9 f10 f11 f12 |
    | esc | ` |
- ``` → `esc`
